To run the Greed Game program:

1. javac Main.java

2. java Main


Controls:

There is a provided list of possible moves before each play

1: down and left
2. down
3: down and right
4: left
6: right
7: up and left
8: up
9: up and right
0: to exit from game

Scoring: the score for each round is the first number in the direction you move
